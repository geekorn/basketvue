## Build Setup

``` bash
# clone this repo
git clone git@github.com:geekorn/gb_nodejs.git recruit-korneev

# go to path
cd recruit-korneev/

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
